# 💰 Монетизация через рекламу

## 📊 Обзор

В приложении StudySAT реализована система монетизации через рекламу с поддержкой:
- **Google AdSense** - для международной аудитории
- **Яндекс.Директ** - для аудитории из СНГ (включая Казахстан)
- **Прямая продажа рекламных мест** - для прямых рекламодателей

## ✨ Особенности

- ✅ **Премиум пользователи не видят рекламу** - стимул для подписки
- ✅ **Адаптивные рекламные блоки** - автоматическая подстройка под размер экрана
- ✅ **Не навязчивая реклама** - размещена в удобных местах
- ✅ **Легкая интеграция** - готовые компоненты для использования

## 🎯 Где размещена реклама

1. **Главная страница** (`/`) - горизонтальный баннер внизу
2. **Страница практики** (`/practice`) - боковая панель или между вопросами
3. **Страница уроков** (`/lessons`) - можно добавить
4. **Страница флеш-карт** (`/flashcards`) - можно добавить

## 🔧 Настройка Google AdSense

### 1. Регистрация в Google AdSense

1. Перейдите на [Google AdSense](https://www.google.com/adsense)
2. Зарегистрируйте аккаунт
3. Добавьте ваш сайт
4. Получите **Publisher ID** (начинается с `ca-pub-`)
5. Создайте рекламные блоки (Ad Units) для разных страниц

### 2. Настройка переменных окружения

Добавьте в файл `.env`:

```env
# Google AdSense
VITE_GOOGLE_ADSENSE_ID=ca-pub-xxxxxxxxxxxxxxxx
VITE_ADSENSE_SLOT_HOME=1234567890
VITE_ADSENSE_SLOT_PRACTICE=1234567891
VITE_ADSENSE_SLOT_LESSONS=1234567892
```

**Где найти:**
- `VITE_GOOGLE_ADSENSE_ID` - ваш Publisher ID из AdSense
- `VITE_ADSENSE_SLOT_*` - ID рекламных блоков (Ad Unit ID)

### 3. Использование компонента

```tsx
import AdBanner from '@/components/ads/AdBanner';

// В вашем компоненте
<AdBanner 
  slotId={import.meta.env.VITE_ADSENSE_SLOT_HOME}
  format="horizontal"
  className="w-full mt-8"
/>
```

## 🔧 Настройка Яндекс.Директ

### 1. Регистрация в Яндекс.Директ

1. Перейдите на [Яндекс.Директ](https://direct.yandex.ru)
2. Создайте рекламную кампанию
3. Получите **Block ID** для рекламных блоков
4. Настройте таргетинг для вашей аудитории

### 2. Настройка переменных окружения

Добавьте в файл `.env`:

```env
# Яндекс.Директ
VITE_YANDEX_DIRECT_ID=R-xxxxxx-xxx
VITE_YANDEX_BLOCK_HOME=xxxxx
VITE_YANDEX_BLOCK_PRACTICE=xxxxx
```

### 3. Использование компонента

```tsx
import YandexAd from '@/components/ads/YandexAd';

// В вашем компоненте
<YandexAd 
  blockId={import.meta.env.VITE_YANDEX_BLOCK_HOME}
  className="w-full mt-8"
/>
```

## 💵 Прямая продажа рекламных мест

Если у вас есть прямые рекламодатели, используйте компонент `AdPlaceholder`:

```tsx
import AdPlaceholder from '@/components/ads/AdPlaceholder';

<AdPlaceholder 
  title="Рекламное место"
  description="Здесь может быть ваша реклама"
  height="250px"
/>
```

Затем замените его на кастомный рекламный блок с HTML/JS кодом от рекламодателя.

## 📈 Стратегии монетизации

### 1. Freemium модель
- Бесплатные пользователи видят рекламу
- Премиум пользователи без рекламы (стимул для подписки)

### 2. Гибридная модель
- Реклама + подписки = двойной доход
- Реклама покрывает операционные расходы
- Подписки - основной доход

### 3. Оптимизация размещения
- **Высокий трафик страницы** = больше рекламы
- **Низкий трафик** = меньше рекламы, больше фокуса на контенте

## 🎨 Рекомендуемые места для рекламы

### Главная страница
- ✅ Внизу страницы (после основного контента)
- ✅ Между секциями статистики
- ❌ Не в шапке (плохой UX)

### Страница практики
- ✅ Боковая панель (если есть)
- ✅ После каждого 5-го вопроса
- ✅ В конце сессии практики
- ❌ Не между вопросом и ответом

### Страница уроков
- ✅ В начале урока (баннер)
- ✅ В конце урока
- ❌ Не в середине урока (прерывает обучение)

## 📊 Ожидаемый доход

### Google AdSense
- **CPC** (цена за клик): $0.10 - $2.00 (зависит от ниши)
- **CPM** (цена за 1000 показов): $1 - $10
- **CTR** (кликабельность): 0.5% - 3%

### Яндекс.Директ
- **CPC**: 10 - 200 тенге (зависит от ниши)
- **CPM**: 50 - 500 тенге
- **CTR**: 0.3% - 2.5%

### Пример расчета
При 10,000 посетителей в месяц:
- Показы рекламы: ~30,000 (3 рекламных блока на пользователя)
- Клики (CTR 1%): ~300 кликов
- Доход (CPC $0.50): ~$150/месяц

## ⚙️ Технические детали

### Компоненты рекламы

1. **AdBanner** - Google AdSense баннер
   - Автоматически скрывается для премиум пользователей
   - Адаптивный дизайн
   - Поддержка разных форматов

2. **YandexAd** - Яндекс.Директ блок
   - Автоматически скрывается для премиум пользователей
   - Нативная интеграция

3. **AdPlaceholder** - Заглушка для прямой рекламы
   - Показывается когда рекламные сети не настроены
   - Можно использовать для продажи мест напрямую

### Проверка премиум статуса

Все компоненты автоматически проверяют подписку пользователя:
- Если пользователь имеет активную подписку (не free) - реклама не показывается
- Если пользователь не авторизован или имеет free план - реклама показывается

## 🚀 Быстрый старт

1. **Выберите рекламную сеть:**
   - Google AdSense (международная)
   - Яндекс.Директ (СНГ)
   - Обе (максимальный охват)

2. **Зарегистрируйтесь** в выбранной сети

3. **Добавьте переменные окружения** в `.env`

4. **Реклама автоматически появится** на страницах

5. **Начните зарабатывать!** 💰

## 📝 Примеры интеграции

### Добавление рекламы на новую страницу

```tsx
import AdBanner from '@/components/ads/AdBanner';

const MyPage = () => {
  return (
    <div>
      {/* Ваш контент */}
      
      {/* Рекламный блок */}
      <div className="mt-8">
        <AdBanner 
          slotId={import.meta.env.VITE_ADSENSE_SLOT_MY_PAGE}
          format="horizontal"
        />
      </div>
    </div>
  );
};
```

### Комбинирование рекламных сетей

```tsx
import AdBanner from '@/components/ads/AdBanner';
import YandexAd from '@/components/ads/YandexAd';

<div className="space-y-4">
  <AdBanner slotId={import.meta.env.VITE_ADSENSE_SLOT_HOME} />
  <YandexAd blockId={import.meta.env.VITE_YANDEX_BLOCK_HOME} />
</div>
```

## ⚠️ Важные замечания

1. **Не перегружайте страницу рекламой** - это ухудшает UX
2. **Следите за производительностью** - реклама может замедлять загрузку
3. **Соблюдайте политику рекламных сетей** - нарушение = бан
4. **Тестируйте на разных устройствах** - реклама должна быть адаптивной
5. **Мониторьте доход** - анализируйте эффективность размещения

## 🔒 Безопасность

- Реклама загружается через официальные скрипты
- Проверка на премиум статус выполняется на клиенте
- Рекламные сети имеют встроенную защиту от мошенничества
